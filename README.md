# Rental CRM

장기렌트 상담 업무를 위한 CRM 프로그램.

## 목적

Word 문서와 Windows 폴더를 이용한 상담 관리 방식을 하나의 프로그램으로 통합한다.

## 주요 기능

- 고객 관리
- 상담 기록
- 첨부파일 관리
- 계약 진행 상태
- 검색
- 대시보드

## 기술 스택

Backend
- ASP.NET Core

Frontend
- React + TypeScript

Database
- SQLite

Storage
- Local File System

## 문서

프로젝트 관련 문서는 docs 폴더를 참고한다.
