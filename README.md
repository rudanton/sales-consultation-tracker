# Consult Note

장기렌트 상담 업무를 위한 개인용 상담 기록 및 파일 관리 프로그램.

Consult Note는 회사 CRM/DB를 대체하는 프로그램이 아니다. 회사의 고객 문의 DB는 기존처럼 사용하고, 상담원이 개인 Windows PC에서 Word 문서와 고객별 폴더로 관리하던 상담 기록, 견적 이미지, 면허증, 계약 관련 파일을 더 빠르게 검색하고 열람하기 위한 보조 도구이다.

## 목표

- 고객별 상담 기록을 빠르게 작성하고 조회한다.
- 고객명, 전화번호, 관심 차량으로 고객을 빠르게 검색한다.
- 고객별 견적 이미지와 첨부파일을 한 화면에서 확인한다.
- Word와 Windows 탐색기를 오가는 시간을 줄인다.
- 압축 해제 후 실행 파일만으로 사용할 수 있게 배포한다.

## 대상 사용자

- 장기렌트 상담원 1인 사용
- 개인 Windows PC에서 로컬 실행
- 데이터는 사용자 PC에 저장
- 중앙 서버나 다중 사용자 동기화는 MVP 범위에 포함하지 않음

## MVP 주요 기능

- 고객 생성, 수정, 목록 조회, 상세 조회
- 고객명, 전화번호, 관심 차량, 상담 메모, 견적 차량명 검색
- 최근 상담일 기준 고객 정렬
- 상담 기록 추가 및 타임라인 조회
- 견적 이미지 추가, 목록 조회, 더블클릭 열기
- 첨부파일 추가, 목록 조회, 더블클릭 열기
- SQLite 기반 로컬 데이터 저장
- 고객별 로컬 파일 저장
- Windows 실행 파일 배포

## 기술 스택

- C#
- WPF
- SQLite
- Entity Framework Core
- Local File System

## 저장 방식

파일 자체는 DB에 저장하지 않는다. DB에는 파일 경로와 검색용 메타데이터만 저장하고, 실제 파일은 로컬 파일 시스템에 저장한다.

예상 저장 구조:

```text
ConsultNote/
├── ConsultNote.exe
├── consultnote.db
├── storage/
│   └── customers/
├── backup/
└── logs/
```

고객별 파일 구조:

```text
storage/
└── customers/
    └── {customerId}/
        ├── estimates/
        └── attachments/
```

## 개발 원칙

- MVP 범위를 벗어나는 기능을 먼저 만들지 않는다.
- 로그인, 권한, 서버 기능을 임의로 추가하지 않는다.
- 파일 바이너리를 SQLite에 직접 저장하지 않는다.
- 사용자가 입력해야 하는 항목은 최소화한다.
- Word와 탐색기를 대체하는 속도와 편의성을 최우선으로 한다.
- 일반 사용자가 개발 도구 없이 실행할 수 있어야 한다.

## MVP 제외 항목

- 로그인, 회원가입, 사용자 권한 관리
- 다중 사용자 기능
- 회사 DB 직접 연동
- 서버 배포, 웹 배포, 모바일 지원
- 실시간 동기화
- 회계/정산, 문자 발송, 전자계약 기능
- 자동 백업, 통계, 대시보드, 다크모드
