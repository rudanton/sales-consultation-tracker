# Coding Guidelines

General

- Use C#
- Use .NET 9
- Use Entity Framework Core
- Use SQLite

Naming

- PascalCase for classes
- camelCase for variables

Functions

- One responsibility
- Less than 50 lines if possible

Database

- Use Repository Pattern

Logging

- Use ILogger

Comments

- Only when necessary

Validation

- Server-side validation required

Avoid

- Duplicate code
- Static mutable state

Files

One class per file.
