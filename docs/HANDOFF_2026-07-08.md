## Current Decisions

- Search is intentionally limited to customer name, phone number, and vehicle/model.
- Consultation record full-text search is not needed.
- Customer file name search is not needed.
- Customer file PDF merge/export is not worth implementing for now.
- Automatic in-app binary replacement is not implemented yet.
  - Current update feature only checks the latest GitHub Release and opens the Release page.
- Backup automation is not planned for now.
  - If PC failure/blue screen risk matters, manually copy `consultnote.db` and `storage/`.
