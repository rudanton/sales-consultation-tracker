## 요청사항
견적 업로드시 차량 종류 설정 -> 이건 차량 자료(마이그레이션 문제)랑 연결될 듯
고객 추가 단축키 있으면 좋겠음.(무난한 건 ctrl+n 이지 않을까.)
양식 오브젝트간 위아래 마진이 너무 큼. 간격이 너무 넓음
고객 추가시 이름 인풋필드에 focus
다크모드 있으면 좋겠음.
ctrl+f 시에 검색창 포커스
즐겨찾기 더 눈에 띄게(unselected시 회색 말고 기본 색깔 계열)
파일 업로드 시 작성중인 기록 날아감
종료시 트레이로 숨길것인지 종료할 건지 물어보기
상담 기록 작성 후 tab시 '저장'버튼으로 focus 되게 : 상담 기록 -> tab -> enter ->  저장(ctrl+enter 와 같은 역할)
양식 기록 중 저장시 상담 기록 날아감
파일 업로드에 클립보드(이미지) 붙여넣기 기능
상담 기록 저장 후에는 상담 상태는 상담중(기본값)으로 변경
상담 기록 export 위치는 파일 저장되는 위치와 동일하게
상담 내역 export가 양식만 export 되는데, 모든 상담 기록이 다 되게
왼쪽 고객목록에 차종 메모 <<  여기에 가운데 점 지우고 공백으로 구분 or 세로선(|) 같은 거.
메모가 비어있으면 - 말고 빈 칸
상담 기록(이미 저장된 기록) 블럭 -> 복붙 가능하도록
Git Release 기능
부재 시 n일 부재 업데이트가 제대로 안 됨. 새로 부재 상담 기록시 0일로 업데이트 되는 건 아닌지 확인
