- 명의자 정보는 상담 양식의 특이사항 아래에 있으며, `고객과 동일` 체크 시 고객 이름/연락처가 채워지고 체크 해제 시 빈 값으로 돌아갑니다.

## Recent UI Decisions

- 상단의 `고객 저장` 명칭은 `기본 정보 저장`으로 정리했습니다.
- 상담 기록 영역은 상태 변경과 상담 저장을 같은 블록에서 처리합니다.
- 양식 하단 버튼은 `차량 찾기`, `양식 저장`, `양식 복사` 3개로 유지합니다.
- `양식 복사`는 현재 입력된 상담 양식을 클립보드에 복사하고 `logs/exports/`에도 txt로 보조 저장합니다.
- 오른쪽 파일 영역은 열린 상태에서 `파일` 제목 오른쪽에 접기 버튼이 있고, 닫힌 상태에서는 작은 `파일 >` 패널로 남습니다.
- 전화번호는 화면 표시와 복사 시 `000-0000-0000` 형태로 포맷됩니다.
